# Carpeta `data/`

Aquí se ubicaría el dataset utilizado en este proyecto: **Employee_churn.csv**.  

⚠️ El archivo CSV fue aportado por una institución privada con fines educativos, por lo que **no se incluye en este repositorio público**.   

Si se desea reproducir el análisis, puede sustituirse por datasets equivalentes de rotación de empleados disponibles en fuentes públicas, como por ejemplo:  
- [Kaggle HR Analytics Employee Churn](https://www.kaggle.com/datasets/giripujar/hr-analytics)
